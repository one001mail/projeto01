# Logging & Retention

## Logger

- **Library:** Pino (`src/infra/logging/logger.ts`).
- **Format:** JSON in production, `pino-pretty` in dev when `PRETTY_LOGS=true`.
- **Per-request fields:** `req.id` (UUIDv4 generated by Fastify), `req.method`,
  `req.url`, `res.statusCode`, `responseTime`.
- **Redaction:** `LOG_REDACT_PATHS` (Pino path syntax). Default removes
  `req.headers.authorization` and `req.headers.cookie`.
- **Sampling:** none today — at preview volume every request is logged.

## Audit log

- **Source:** `src/api/http/middlewares/audit-log.middleware.ts` —
  `onResponse` hook that fires for every `2xx` mutating response
  (`POST/PUT/PATCH/DELETE`).
- **Adapters:** `pg-audit-log.store.ts` (Postgres), `in-memory-audit-log.store.ts`
  (sandbox).
- **Schema:** `(id, request_id, method, path, status_code, redacted_payload,
  inserted_at)`.
- **Redaction:** dot-path list `AUDIT_REDACT_FIELDS`. The redactor itself is
  pure (`src/shared/application/redaction.ts`) and 100 % unit-tested.

## Retention

| Data                          | Default | Configured by                |
| ----------------------------- | ------- | ---------------------------- |
| HTTP logs                     | 7 days  | `LOG_RETENTION_DAYS`         |
| Audit-log rows                | 30 days | (TODO scheduled cleanup job) |
| Idempotency cache             | 24 h    | `IDEMPOTENCY_TTL_SECONDS`    |
| Outbox dispatched rows        | 7 days  | (TODO archival worker)       |

Cleanup happens in a background worker (`src/infra/queue/scheduler-worker.ts`)
that runs on the same loop as the outbox dispatcher when `WORKERS_ENABLED=true`.

## Where to look in production

- Application logs: stdout of the Node process (collected by the runtime).
- Audit rows: `audit_log` table in PG.
- Idempotency cache: `idempotency_cache` table in PG (or Redis when configured).
- Workers heartbeat: log line `outbox-dispatcher tick` every poll interval.

## Where to look in sandbox / preview

- `/var/log/supervisor/backend.err.log` (uvicorn shim + Fastify Pino interleaved).
- `/var/log/supervisor/backend.out.log`.
